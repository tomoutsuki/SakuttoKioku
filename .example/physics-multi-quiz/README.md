# Physics Multi-Quiz Fixture

This folder contains a multi-quiz physics package for testing ZIP imports in Sakutto Kioku.

Contents:

- `mechanics/quiz.json`
- `optics/quiz.json`
- `thermodynamics/quiz.json`
- `electromagnetism/quiz.json`

The `mechanics` and `optics` quizzes also include simple SVG assets to test image extraction from ZIP packages.

Import target:

- `.example/physics-multi-quiz.zip`
